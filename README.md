# Tic Tac Toe (Ad-Free Family Edition)

A simple **Tic Tac Toe** game built with **Vanilla JavaScript, HTML, and CSS**, originally created for my family so they could play without ads or distractions commonly found in mobile apps.

What started as a small personal project grew into a learning playground where I experimented with new concepts, game mechanics, and even turned it into a **Progressive Web App (PWA)**.
